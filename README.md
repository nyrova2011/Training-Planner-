## Описание
Это простое GUI-приложение для планирования и учета тренировок. Пользователь может добавлять тренировки с указанием даты, типа и длительности. Также реализована фильтрация по типу и дате, а все данные сохраняются в JSON-файл для последующего использования.  

## Основные функции
- Ввод данных о тренировке (дата, тип, длительность)
- Добавление тренировки в таблицу
- Фильтрация по типу тренировки
- Фильтрация по дате
- Сброс фильтров
- Сохранение данных в JSON и загрузка при запуске

1. Склонируйте или скачайте репозиторий.
2. Убедитесь, что у вас установлен Python 3.
3. Запустите файл `training_planner.py` через командную строку:

```bash
